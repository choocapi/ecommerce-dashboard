# ACB Computer — Dashboard

Internal admin panel for managing the ACB Computer ecommerce platform.

🌐 **Live:** [dashboard.acbcomputer.store](https://dashboard.acbcomputer.store)

## Features

- **Inventory & product management** — Full product lifecycle: import → list → sell → deliver → return
- **Order management** — Track, process, and export/print invoices
- **Analytics** — Revenue, orders, sales, product performance, inventory reports
- **Customer & staff management** — Role-based permissions for staff
- **Returns & discounts** — Handle return requests and coupon codes
- **Flexible UI** — Rich filters and layouts to optimize daily workflows

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js |
| UI | Shadcn UI, Tailwind CSS |
| State | Zustand |

## Getting Started

```bash
git clone https://github.com/choocapi/ecommerce-dashboard.git
cd ecommerce-dashboard
pnpm install
pnpm dev
```

## Project Navigation

This is part of a 3-repo project:

| Repo | Description |
|------|-------------|
| [Storefront](https://github.com/choocapi/ecommerce-storefront) | Customer-facing store |
| ▶ **Dashboard** (this repo) | Admin & staff management panel |
| [Backend](https://github.com/choocapi/ecommerce-backend) | API server |
